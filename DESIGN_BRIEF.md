# ExcelBridge — 设计概要

## 软件定位

**一句话**：可视化 Excel 数据批量匹配迁移工具 — 用 4 次点击替代 VLOOKUP 公式。

**解决的问题**：从多个分散的 Excel/xls 文件中，按关键字段自动匹配并汇总数据到目标模板。全程可视化操作，无需写公式或代码。

## 核心功能流程

```
选择源文件夹 ──→ 点击源文件中「要复制的值」──→ 点击「匹配键」
                                      │
           选择目标模板 ──→ 点击目标「匹配键」──→ 点击「粘贴到」
                                      │
                              预览确认（逐条勾选）──→ 写入结果
```

| 步骤 | 操作 | 视觉反馈 |
|------|------|---------|
| 1 | 选择源文件夹 + 目标模板 | 自动扫描，列表显示 |
| 2 | 点击源文件中「要复制的值」 | 🔵 蓝色选中 |
| 3 | 点击源文件中「匹配键」 | 🟠 橙色选中 |
| 4 | 点击目标中「匹配键」 | 🟢 绿色选中 |
| 5 | 点击目标中「粘贴到」 | 🟣 紫色选中 |
| 6 | 预览匹配结果 | 弹窗展示所有数据，可勾选 |
| 7 | 确认写入 | 后台执行，实时日志 |

## 技术特征

- **双格式**：`.xlsx` + `.xls`（含老格式兼容）
- **保留样式**：写入时保留目标模板原有格式
- **智能过滤**：自动跳过表头行（不限表头位置）
- **规则存取**：映射规则可保存/加载为 JSON
- **后台写入**：不阻塞界面
- **跨平台**：Linux 开发 / Windows 7+ 部署
- **零依赖**：仅需 Python 3.8 + openpyxl + xlrd + customtkinter + Pillow

## 设计语言

### 布局

三栏结构（VSCode/Notion 风格）：
- **左侧栏 250px**：文件夹选择 + 文件列表 + 规则操作
- **中间面板**：上下分屏预览（源文件 / 目标模板）
- **右侧栏 270px**：操作指引 + 映射规则 + 执行按钮 + 内嵌日志

## Logo 设计建议

### 核心隐喻

**四色桥接** — 四个彩色方块用连线串起，代表 4 次点击完成数据迁移

### 推荐方向

**方案 A：四色箭头桥**

```
  [源文件] ──🔵──🟠──→ 🟢──🟣──→ [目标文件]
   图标      复制  匹配    匹配  粘贴    图标
```

简化为 logo：两个圆角文件图标，中间一条四色渐变连线。

**方案 B：魔方拼图**

四个彩色方块（蓝橙绿紫）拼成一个正方形，象征把分散的数据块精准拼合。

**方案 C：钥匙 + 表格**

一把钥匙（匹配键）插入表格单元格，数据从左侧流入右侧。

### 配色参考

```
四色渐变: #3B82F6 → #F59E0B → #10B981 → #8B5CF6
文件图标: #2563EB（源）/ #10B981（目标）
背景: #F7F9FC 或透明
```
